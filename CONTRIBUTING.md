# Contributing to Terminos

## Language Guidelines

### Communication
- **User Questions**: Chinese (中文), Japanese (日本語), or English are all supported
- **AI Responses**: All AI assistant responses will be in English
- **Code Comments**: All code comments must be in English
- **Documentation**: All documentation should be in English

### Code Style
- Use descriptive English names for variables, functions, and classes
- Write all comments in English
- Include English docstrings for all public APIs
- Use English for error messages and logging

### Documentation
- Write all technical documentation in English
- Include English examples in documentation
- Use English terminology consistently

## Development Setup

1. Install Rust toolchain
2. Configure Cursor AI with the provided `.cursorrules`
3. Follow the language guidelines above
4. Run tests before submitting changes

## AI Assistant Usage

When using Cursor AI:
- Ask questions in Chinese (中文), Japanese (日本語), or English
- Expect responses in English
- All generated code will have English comments
- Documentation will be generated in English

## Multilingual Support

The AI assistant supports:
- **Chinese (中文)**: 可以用中文提问，获得英文回答
- **Japanese (日本語)**: 日本語で質問でき、英語で回答を得られます
- **English**: Ask questions in English, receive responses in English

All responses, code comments, and documentation will be consistently in English for maintainability and international collaboration.
